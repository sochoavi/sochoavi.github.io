# GitHub Pages Site

This is a simple static personal site for GitHub Pages.

## Customize

Edit `index.html` to refine the biography, publication list, links, or contact information.

The homepage uses `assets/avishek-majumder.jpg` for the portrait.

## Publish On GitHub Pages

1. Create a GitHub repository named `<your-username>.github.io`.
2. Add these files to that repository.
3. Push to GitHub.
4. Open `https://<your-username>.github.io`.

For a project page instead of a user page, push these files to any repository and enable Pages from `Settings > Pages`, using the `main` branch and root folder.
